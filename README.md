# 🚀 Social Media Automation Scheduler

**Social Media Automation Scheduler** is a full-stack AI-powered social media management platform that helps creators, marketers, and businesses automate their entire social media workflow. Create content, generate AI-powered captions and images, schedule posts across multiple platforms, and manage all your social activity from a single dashboard.

## 🚀 Installation

### Clone Repository

```bash
git clone https://github.com/your-username/social-media-automation-scheduler.git
cd social-media-automation-scheduler
```

---

### Frontend Setup

```bash
cd client

npm install

cp .env.example .env

npm run dev
```

---

### Backend Setup

```bash
cd server

npm install

cp .env.example .env

npm run build

npm start
```
